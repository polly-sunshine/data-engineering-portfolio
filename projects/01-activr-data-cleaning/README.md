# ActiVR Data Cleaning Project

This is a SQL data cleaning and analysis project completed as part of my DataCamp learning path.

## Project Overview

The goal of this project was to clean and analyze event data for ActiVR, a virtual reality company.  
The project focused on fixing missing values, joining event and game data, and preparing insights for the sales team.

## Skills Practiced

- SQL
- Data cleaning
- JOINs
- Filtering
- Aggregations
- GROUP BY
- COUNT DISTINCT
- Creating analysis-ready tables

## Project Tasks

- Identified and fixed missing `game_id` values in the events table.
- Found users who participated in biking-related game events.
- Counted unique users by `game_type` and `game_id`.
- Prepared clean query outputs for business analysis.
